Frequently Asked Questions

Return


    What is OneeChan?
      OneeChan is a userstyle for 4chan.org, made by myself, seaweed (!Seaweed9zY). It features many options to customize 4chan to your liking, including many different color schemes. The main change to 4chan you'd notice is the rearranged layout of 4chan, with a sidebar for easy access to the post form. It can of course be arranged to look almost identical to the default 4chan, so you won't miss out on benefits just because you prefer the regular look.
    That just sounds like AppChan. What's the difference?
      OneeChan was actually made as a fork of AppChan. I was a developer for AppChan for a few months and the main person fixing bugs and making commits during that time, and I left the project due to disputes about controversial options that just added in more bugs. About a month later, AppChan was untouched, as well as the bugs related to my departure, and AppChan's creator, Zixaphir, decided that its hiatus was better than my return. I was disheartened as a former AppChan developer and as an AppChan user, so I took matters into my own hands and forked the project instead of waiting any longer (it was licensed as WTFPL at the time). OneeChan's creation and gaining popularity has led to Zixaphir's return to the AppChan project, so now both exist with similar goals but different directions.

      Specifically, what sets OneeChan apart are some new color schemes, new mascots, new options, and a "toolbox" of icons at the top to save screen real estate. For specifics, see the changelog.
    What does "OneeChan" mean?
      Its Japanese for "older sister." Take from that what you will. In Japanese its pronounced oh-nay-chahn (ay as in neigh, ah as in gah!), but some who don't know Japanese pronounce it oh-nee-chan (ee as in knee, a as in hat).
    AppChan has y. Why doesn't OneeChan?
      I do not look at AppChan, and I never intend to use anything in AppChan aside from what existed at the time of the fork. I intend to keep it this way, because while AppChan may borrow from OneeChan, I feel that the projects should be keep separated to maintain separate identities, and -- for you, the user -- provide two very different options, instead of making you think, "...They're the exact same thing. Which one do I pick?" If there's something AppChan does differently that you as an OneeChan user would like me to implement, feel free to let me know.
    I posted x issue on 4chan, why won't you fix it?
      I can't be monitoring 4chan all the time for people who might post an issue there. If you want to report an issue, you should do it on the issues tracker so that I can be easily notified of it whenever I sign on, and you can be notified of responses to it and when its resolved. Registering for Github does not require an email address, but if its still an issue you can report issues to me anonymously here.
    I made a mascot / color scheme! How do I send it to you to put on OneeChan?
      You can send it to me as a private message on Github ("Notifications" -> "Compose Message"), or through the email function on Github ("Message" button on my profile). Use a site like imgur to upload mascots, and gist or pastebin to upload color schemes. Just note that nothing is of course guaranteed to be put into OneeChan, but I am pretty open-minded and more options are never a bad thing.


OneeChan on Firefox

Return

  • Install Stylish OR Scriptish if you don't have either already. You only need one of them.
  • Go to the official OneeChan page.
  • Read through the different options and select all the options you'd like to have, or alternatively let OneeChan choose the defaults for you.
  • Stylish: Click the "Install with Stylish" button in the green box underneath the options.
  • Scriptish: Left click (don't right click) the "Install as UserJS" link in the green box underneath the options. When the new page loads, a Scriptish dialog should prompt you to install.

    That's all! You should now have a working OneeChan installation.

    Enjoy.

OneeChan on Opera

Return

    Notice: It's recommended that you disable 'smooth scrolling' in your Opera preferences. The option doesn't work very well with userscripts that add a lot of CSS.

  • Go to the official OneeChan page.
  • Read through the different options and select all the options you'd like to have, or alternatively let OneeChan choose the defaults for you.
  • Left click (don't right click) the "Install as User JS" link in the green box underneath the options.
  • Opera -> Save Page As...
    Now put the file into the same folder you put 4chan X in. If you don't have that, make a new folder, name it anything you want like "UserJS", and save the OneeChan file there.
  • Now you just need to make sure that your UserJS folder is properly loaded by Opera:
    Opera -> Settings -> Preferences -> Advanced -> Content -> JavaScript Options
  • Under the window that pops up, choose the folder you put OneeChan into under "User JavaScript folder". Setting it as a site specific preference will not work properly as OneeChan uses very specific website rules for different 4chan regexps, domains, and url-prefixes.

    • That's all! You should now have a working OneeChan installation.

    Enjoy.

04. Enable Sidebar?
This one's prety obvious. It enables the sidebar on the right side (or left side) of the screen, where board names, the toolbox, the logo, mascots, etc., all go.
05. Margin Style (not the sidebar)
This is how big the other side opposite the sidebar is.
"No Left Margin" means replies will reach the edge of the window.
"Small Left Margin" leaves a little bit of space -- this is what 4chan uses by default.
"Medium Left Margin" makes it slightly large than default, but not too much larger.
"Large Left Margin" is double the margin space.
"Completely Centered" as it says instead centers the replies within the window. This is the only option that changes the side with the sidebar, as it makes it so the margins on both sides are equal instead of aligning it with the sidebar.
06. Navigation Type
You probably know on 4chan about the top navigation links and the pages links. This option lets you fix them to the window so that you always have them available as you scroll through threads. The options vary by what's positioned where, and even lets you only have one stickied. If you want you can even disable the header completely and just use the slideout navigation in the toolbox.
07. Pages Location
Instead of displaying page links horizontally, you can display them vertically along the side of the page. This WILL override the "Navigation Type" option's rules for the page links.
08. Board Name + Banner
"/g/ - Technology" would be the board name, and the logo would be the often animated GIFs that usually say 4chan on them. If you have it set to just "Name", then its positioned over your mascot by the post form. Anything else is positioned towards the top of the window, closer to what it is by default. These are all shown in the sidebar.
09. Custom Board Names
This sets pseudo-elements to replace the board names. Instead of, for example, it saying "/g/ - Technology", it could say "/g/entoo". You have to go in and edit this one yourself to get it working the way you want. Search in your userstyle by opening up the userscript file or browsing through Stylish's options and search for "/g/ - Technology." The "content" part, which by default does say "/g/ - Technology," you want to change to whatever you want it to say. You can copy and paste the section and change the board in the link to create new rules for other boards' names as well.
10. Quick Reply / Post form
4chan X now controls normal post form visibility, OneeChan does not hide it. These options apply to both the post form and the quick reply.

"Fixed" - The post form / quick reply is fixed in your sidebar and always visible, ready for you to make a new thread or post.
"Slideout" - Half of the post form is hidden in the side of the window until you hover over it.
"Tabbed Slideout" - A little prettier than the Slideout option, this makes it so all that's visible until hover is a small tab that says "Quick Reply" or "Post Form".
"Fade" - This works similar to "Fixed," except that it makes the post form slightly transparent until you hover over it.
"Floating QR Only" - This keeps the 4chan X quick reply floating like it would be without OneeChan.
11. Name/Email/Subject Style
"Compact" - This puts the name, email, and subject input fields on the same row of the quick reply, making them smaller.
"Non-Compact" - This puts the input fields on their own row, similar to how 4chan and the old quick reply used to be.
12. Postarea / QR Labels?
These are the labels for the input fields and textareas. They say "Name" in the name field, "Email" in the email field, etc.
13. Expand Textarea on Focus?
When you click the textarea, it will expand to a large textarea giving you more room to type.

16. Reply Width
You can either set this option to Normal so that reply width depends on how much text is in the reply, or you can set it to "Make replies as wide as the page" which makes the replies take up all available space.
17. Rounded Corners
If you'll notice around this very box you'll see the corners are not sharp but slightly rounded. That's all this option does. You can enable it, enable it with completely circular checkboxes, or keep 'em square and sharp.

20. Expanded Filenames
4chan by default cuts off some of the file name to avoid it getting too long. By enabling this option, you'll simply be shown the entire file name. If you have "Rice filenames" enabled in 4chan X, this OneeChan option will be ignored.
21 Centered Images in New Tabs (Chrome only)
When you open images in a new tab, via clicking them without Image Expanding enabled or right click -> Open in new tab, this will center the image on the page. Normally they appear in the top left corner. To avoid scrollbars hiding entirely, this will have a horizontal scrollbar that's always visible. As of Firefox 11, Firefox does this on its own, so OneeChan will ignore this option on Firefox.
22. Custom Scrollbars
This option changes the colors of the scrollbars to match that of your chosen color scheme.
23. Image Hover Fixes
When you hover over a link in Firefox or Chrome, a small ~20px popup appears at the bottom that shows the URL of the link you're hovering. Now when image hover is enabled in 4chan X, the images get covered up by that popup. Another issue is that large images get shown off the page. The settings for this option are to provide solutions for both of these issues, just one, or none at all.
24. Quote Backlinks (Quoted By:)
Backlinks in 4chan X, which come from the /b/ackwash userscript, normally appear next to the post number. You can change the position to other corners of the reply as well as add text that that says "Quoted By:" before the backlinks, with this option. It can look out of place with quote previews, and will have usability issues with inlined replies that are designed with the backlinks placement in their normal state, so use it with caution.
25. Moot's Announcements
When moot makes an announcement, without OneeChan, the text is read and it appears at the top of the page. You can move it to a toolbox icon that displays when moot makes an announcement and shows the announcement on hover, or you can disable it entirely.
26. Slideout Watcher?
One criticism of the thread watcher in 4chan X is that its not always needed and can cover up posts when it gets too long. This solves that by placing an icon in the toolbox that shows the thread watcher when you hover over it. You can of course disable it.
27. Slideout Navigation?
Shows a small button in the corner with a compass icon that displays navigation links in a compact way.

"List" - Shows the full names of boards and makes them easily clickable.
"Compact" - Only shows the board letter.
28. Current Board Highlighting
This will make the board letter of the board you're on bold in the navigation bar.
29. reCaptcha Opacity
With this you can set how transparent the post form reCaptcha is, so the white isn't as powerful on darker color schemes.
30. Hide Name Field Content?
For tripcode users who don't want their tripcode accidentally getting into screenshots of their browser, this hides the content of the name field until you hover it.
31. Slimmer Replies
This option controls the padding INSIDE replies.
32. Underline Links
If you like things underlined, like regular links, or the [Reply] link, this option will underline them for you.
33. Margin Between Replies
OneeChan by default makes the space between replies smaller than it is by default. If you're one of those who want even less space between replies, the "None" option will make that happen. If you're trying to set up an almost vanilla 4chan look, the "Normal" option is the way to go.
34. Filtered Backlinks
4chan X creates strikethrough links to filtered posts with no way to hide the links entirely, this does just that.
35. Unreverse Unicode Names?
Some people reverse their names, which also reverses every part of their post making it hard to reply to them. This undoes that.
36. Sidebar Location
Simply put, you can leave your whole sidebar in its default position on the right side, or you can move it to the left side!
